To compile and run the Student Grading System:

1. Prerequisites:
   - GCC compiler (MinGW for Windows, or gcc on Linux/macOS)

2. Compilation Instructions:
   - Windows (Command Prompt or PowerShell):
     ```
     gcc -o student_grading_system student_grading_system.c
     ```
     
   - Linux/macOS (Terminal):
     ```
     gcc -o student_grading_system student_grading_system.c
     ```

3. Running the Program:
   - Windows:
     ```
     student_grading_system
     ```
     
   - Linux/macOS:
     ```
     ./student_grading_system
     ```

4. System Features:
   - Add, list, search, update, and delete student records
   - Calculate student averages and letter grades
   - Generate class reports with statistics
   - Save and load data from/to text files

5. Notes:
   - The system stores student data in "students.txt"
   - Class reports are saved to "class_report.txt"
